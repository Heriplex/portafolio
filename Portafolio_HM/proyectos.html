Implementación de Servidor Linux con Bacula

Configuración e instalación de un servidor Linux con el software Bacula para realizar respaldos automáticos de los datos críticos de la empresa. Se estableció una política de respaldo personalizada y se automatizó el proceso para garantizar la integridad y disponibilidad de la información.

Tecnologías utilizadas: Linux (Ubuntu Server), Bacula, scripting (bash)

Levantamiento de Red de Datos para Empresa Automotriz

Diseño e implementación de una red de datos eficiente y segura para una empresa automotriz. Se realizó un análisis de las necesidades de conectividad, se configuraron switches, routers y puntos de acceso Wi-Fi para garantizar una cobertura óptima en todo el establecimiento

Tecnologías utilizadas: Cisco networking equipment, VLANs, VPN, firewall.

Sistema de Puntos de Venta y Red de Infraestructura para Restaurante

Desarrollo e implementación de un sistema integral de puntos de venta para un restaurante, incluyendo la configuración de impresoras de red, terminales de punto de venta y un servidor central.



Tecnologías utilizadas: Windows server 2019, Sistema POS y puntos de venta, red de datos
